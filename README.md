## 🌟 What is MARIA?

**MARIA (Magical AI Reading & Instruction Assistant)** is an interactive AI-powered learning platform designed to make education:

- 🎨 Fun & engaging  
- 📖 Curriculum-driven  
- 🧠 Personalized with teacher personas  
- 💬 Conversational & adaptive  

Built using **LLMs + RAG (Retrieval Augmented Generation)**, MARIA creates a magical classroom experience for children.

---

## 🧩 Key Features

✨ **Teacher Personas**  
Choose from multiple teaching styles (friendly, strict, revision-focused, etc.)

📚 **Curriculum-Based Learning**  
Supports structured learning across:
- Boards
- Classes
- Subjects
- Topics

🧠 **RAG-powered Knowledge System**  
- FAISS vector search
- Context-aware responses
- Curriculum grounding

💬 **Interactive Chat Learning**
- Child-friendly explanations
- Emoji-rich responses 🎉
- Adaptive conversation flow

🚦 **Smart Decision Engine**
- Curriculum vs Question vs Chitchat detection
- Safe & moderated responses for kids

🎮 **Gamified Experience**
- Progress tracking 📊  
- Topic progression 🎯  
- Confetti celebrations 🎊

## ⚙️ How It Works (Colab Steps)

1. 🛠️ Install dependencies  
2. 📥 Download knowledge base & curriculum  
3. 🤖 Load model (**Qwen3-4B INT4**)  
4. 🧩 Initialize RAG + helpers  
5. 🎨 Launch Gradio UI  
6. 🚀 Start learning!

---

## 🧠 Tech Stack

- 🤖 **LLM:** Qwen3-4B (INT4 Quantized)
- 🔍 **Embeddings:** Sentence Transformers
- 📦 **Vector DB:** FAISS
- 🧮 **Frameworks:** Transformers, Accelerate, BitsAndBytes
- 🌐 **UI:** Gradio
- 🐍 **Backend:** Python

---
